# Automated Outreach Pipeline

## Overview

This project automates the cold outreach process.

Pipeline:

Company Domain
→ Similar Companies
→ Decision Makers
→ Email Discovery
→ Outreach Email Sending

## Technologies Used

- Python
- Ocean.io
- Prospeo
- EazyReach
- Brevo

## Features

- Automated lead generation
- Contact discovery
- Email verification
- Automated outreach
